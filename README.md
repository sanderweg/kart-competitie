# Kart Competitie met uitslagen bewerken

Nieuw in deze versie:
- op de adminpagina kun je bestaande uitslagen bewerken
- knop `Uitslag bewerken` bij iedere race
- formulier wordt automatisch gevuld met de bestaande gegevens
- knop `Bewerken annuleren`
- opslaan overschrijft de bestaande race in Firebase
